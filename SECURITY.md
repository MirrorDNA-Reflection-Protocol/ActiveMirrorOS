# Security Policy

## Supported Versions
We support the latest release of the MirrorDNA Protocol (v1.x).

## Reporting a Vulnerability

If you discover a security vulnerability within MirrorDNA, Active MirrorOS, or MirrorBrain, please Report it responsibly.

**DO NOT** open a public issue for sensitive vulnerabilities.

### Contact
Email: `security@mirrordna.org` (or contact Paul Desai directly via Twitter/GitHub).

### Policy
- We prioritize user sovereignty and data privacy.
- Vulnerabilities exposing user data or allowing unconsented identity drift are Critical.
- We will acknowledge receipt within 48 hours.
